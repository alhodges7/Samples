Creating a New Resource



To add a new Resource to the system, you must do the following:

1. Logon to Astra (you must be a Librarian).
2. Click the Admin Link on the main site menu.
3. Click the Resource List link on the Index of Admin Functions. The Resource List page will load.
4. At the top of the Resource List page, click the Create New Resource link.
5. In the New Resource Editor editor, fill-out all of the fields that you can. The list of fields is as follows:

Title
Edition
Author
Length
Description
Copies
Skill Level
ISBN10 Number
ISNB13 Number
Url
KeyWords

6. Click the Create button when all relevant fields are done.
