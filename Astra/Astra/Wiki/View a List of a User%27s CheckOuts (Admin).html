View a List of a User's Check Outs (Admin)

To view a list of a User's CheckOuts as an Admin, complete the following steps:

1. Logon to Astra (you must be a Librarian).
2. Click the Admin link on the main site menu.
3. Click the Users List link on the Index of Admin Functions. The Users List page will load.
4. Find the User you want to view in the list.
5. Click the Checkouts link in the History column for that user. The user's Check Out History Page will load.
